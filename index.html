My Work

I'm thrilled to present my work to you. Here are some of my projects showcasing my skills. Please have a look!

Mall Customer segmetation

In this project, I managed the Mall customer dataset from Kaggle, performed EDA to uncover key trends, and used K-Means clustering to segment customers, visualizing results with 2D and 3D scatter plots. This provided strategic business insights into customer demographics and spending behavior.

Iris Dataset Classification

In this project, I used linear regression on the Iris dataset to predict continuous features from sepal and petal measurements, evaluating model performance with MSE and RÂ². I also performed classification using logistic regression and LDA to categorize iris species, assessing effectiveness with accuracy, precision, recall, and confusion matrices.

Handwritten Digit Recognition using MNIST

This Python project implements a custom neural network framework for training on the MNIST dataset, featuring essential layers and managed by a `Sequential` class. It supports training with preset hyperparameters, performs forward and backward passes, updates using gradient descent, evaluates accuracy, and visualizes loss trends, providing a foundational approach to neural network training.

Battle of Neighborhoods

In the "Battle of Neighborhoods" project, we analyzed Toronto's neighborhoods to assist a user in finding a new place similar to their current one. We scraped data from Wikipedia, cleaned it, and used the Foursquare API for venue information. The data was clustered into two groups using k-means, revealing clusters with distinct venue types. The analysis aimed to provide insights into neighborhood similarities based on available amenities.
